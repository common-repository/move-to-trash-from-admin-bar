Move to Trash from Admin Bar plugin adds a Move to Trash Button to delete posts or pages from the WordPress Admin Bar.

== Description ==

Add a Move to Trash Button to delete posts or pages from the WordPress Admin Bar.

Just install the plugin; there are no additional settings to be made.

== Installation ==
The installation is pretty standard:

1. Upload the package contents to to the `/wp-content/plugins/` folder
2. Activate the plugin through the `Plugins` menu in WordPress

There are no additional settings to be made.
